# $NFT - Name Fungible Token

A procedurally generated 3D particle visualization engine built with Next.js and Three.js.

## Features

- Procedurally generated 3D shapes from text input (Shape DNA)
- Real-time particle animation with 25,000+ particles
- Dynamic color palettes generated from text seed
- Interactive controls for customization
- Fully responsive design

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This app is optimized for deployment on [Vercel](https://vercel.com).

Simply push to your GitHub repository and connect it to Vercel for automatic deployments.

## Tech Stack

- Next.js 14
- React 18
- Three.js
- TypeScript
